GOLD VALUATION TRACKER — IPHONE APP PACKAGE
Created from the completed Gold_Valuation_Tracker.xlsx workbook dated June 30, 2026.

WHAT IS INCLUDED
- The 10 completed tracker items are preloaded.
- Starting spot price: $3,434.37 per troy ounce.
- Dealer payout model: 90%.
- Pawn payout model: 75%.
- Full melt, dealer, pawn, and 75% of dealer-estimate scenarios.
- Add, edit, delete, search, filter, and sort gold items.
- Automatic local saving on the phone.
- JSON backup/restore and CSV export.
- Offline use after installation.

HOW TO PUT IT ON AN IPHONE
1. Unzip this package on a computer.
2. Upload the entire unzipped folder to an HTTPS web host. Netlify Drop is a simple option:
   - Sign in to Netlify.
   - Open the site deployment/drop area.
   - Drag the entire unzipped folder onto the upload area.
   - Netlify provides a secure web address.
3. Open that web address in Safari on the iPhone.
4. Tap Share.
5. Tap Add to Home Screen.
6. Tap Add.

IMPORTANT
- iPhone installation requires the app to be hosted on an HTTPS website. Opening index.html directly from Files will not provide full PWA installation/offline behavior.
- The gold spot price is entered manually. Update it in the Dashboard whenever you want a new valuation.
- Data is stored in Safari on that device. Export a JSON backup before clearing website data, replacing the phone, or moving the app to another device.
- Melt and payout values are estimates and do not replace professional testing or a binding buyer offer.
