GOLD TRACKER — WEB APP VERSION 4.1

NEW IN VERSION 4
- Import gold-item data from CSV, XLSX, or XLS files.
- Preview imported rows before saving.
- Choose to append imported items or replace the current item list.
- Download blank CSV and Excel import templates from the Data tab.
- Imported data is saved only in the individual user's browser on that device.
- Existing Version 3 user data is preserved during the update.

SUPPORTED IMPORT COLUMNS
Required:
- Description
- Gross Weight (g)
- Karat, or Purity Override %

Optional:
- Non-Gold Deduction (g)
- Purchase Cost ($)
- Hallmark / Test Notes
- Status

Recognized karats include 24K, 22K, 18K, 14K, and 10K. Common hallmarks such as 999, 916, 750, 585/583, and 417/416 are also recognized.

PUBLISHING
1. Unzip this package into a new empty folder.
2. Upload the contents as a new deployment; do not merge with an older app folder.
3. Open the secure hosted address in Safari, Chrome, Edge, or another modern browser.
4. On iPhone/iPad: Safari > Share > Add to Home Screen.
5. On Android: Chrome menu > Add to Home screen or Install app.

LOCAL STORAGE
- Items and settings automatically save in that browser on that device.
- Export Backup creates a JSON copy for transfer or recovery.
- Clearing browser website data can erase locally saved data.

VALUATION NOTICE
Values are estimates and do not replace professional metal testing or a binding buyer offer.
